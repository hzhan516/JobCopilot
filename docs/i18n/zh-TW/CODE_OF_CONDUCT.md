# 貢獻者公約行為準則

## 我們的承諾

作為成員、貢獻者和領導者，我們承諾讓參與我們社群的每個人都不受騷擾，無論其年齡、體型、可見或不可見的殘疾、種族、性別特徵、性別認同和表達、經驗水平、教育程度、社會經濟地位、國籍、個人外貌、種族、種姓、膚色、宗教或性取向如何。

我們承諾以有助於營造開放、熱情、多元、包容和健康社群的方式行事和互動。

## 我們的標準

有助於為我們的社群營造積極環境的行為示例包括：

- 對他人表現出同理心和善意
- 尊重不同的意見、觀點和經歷
- 給予並優雅地接受建設性反饋
- 承擔責任並向受我們錯誤影響的人道歉，並從中吸取教訓
- 不僅關注對我們個人最有利的事情，還關注對整個社群最有利的事情

不可接受的行為示例包括：

- 使用性化語言或圖像，以及任何形式的性關注或性暗示
- 挑釁、侮辱或貶損性評論，以及人身或政治攻擊
- 公開或私人騷擾
- 在未經明確許可的情況下發布他人的私人資訊，例如物理地址或電子郵件地址
- 其他在專業環境中可能被視為不適當的行為

## 執行責任

社群領導者負責闡明和執行我們可接受行為的標準，並將針對任何他們認為不適當、威脅性、冒犯性或有害的行為採取適當和公平的糾正措施。

社群領導者有權和責任刪除、編輯或拒絕不符合本行為準則的評論、提交、程式碼、wiki 編輯、議題和其他貢獻，並在適當時溝通審核決定的原因。

## 適用範圍

本行為準則適用於所有社群空間，也適用於個人在公共空間正式代表社群時。代表我們社群的示例包括使用官方電子郵件地址、通過官方社交媒體帳戶發布內容，或作為指定代表參加線上或線下活動。

## 執行

可以向負責執行的社群領導者報告辱罵、騷擾或其他不可接受的行為，郵箱為 **conduct@jobcopilot.dev**。
所有投訴都將得到及時和公平的審查和調查。

所有社群領導者都有義務尊重任何事件報告者的隱私和安全。

## 執行指南

社群領導者將遵循這些社群影響指南，以確定他們認為違反本行為準則的任何行為的後果：

### 1. 糾正

**社群影響**：使用不適當的語言或其他被認為不專業或不受歡迎的行為。

**後果**：社群領導者的私人書面警告，明確說明違規的性質並解釋為什麼該行為不適當。可能要求公開道歉。

### 2. 警告

**社群影響**：通過單一事件或一系列行為違規。

**後果**：對持續行為後果的警告。在指定時間段內不得與相關人員互動，包括與執行行為準則的人員進行未經請求的互動。這包括避免在社群空間以及社交媒體等外部渠道中進行互動。違反這些條款可能導致臨時或永久禁令。

### 3. 臨時禁令

**社群影響**：嚴重違反社群標準，包括持續的不當行為。

**後果**：在指定時間段內禁止任何形式的互動或公開交流。在此期間，禁止與相關人員（包括與執行行為準則的人員）進行公開或私人互動。違反這些條款可能導致永久禁令。

### 4. 永久禁令

**社群影響**：表現出違反社群標準的模式，包括持續的不當行為、騷擾個人，或攻擊或貶低某類人群。

**後果**：永久禁止在社群中進行任何形式的公開互動。
